# Place_CTF_PSUTI
Веб площадка для соревнований в ПГУТИ

Проект на Laravel с реализованной регистрацией пользователей для участия и подготовки к соревнованиям и олимпиаде по практической кибербезопасности.
На данный момент платформа имеет админпанель с возможностью регистрацией пользователей, добавлением и проверкой задач.

## Реализованно:
Блокировка доступности задач и результатов мероприятий по времени.
Добавление и редактирование задач жюри через админпанель.
Ручная проверка ответов участников для олимпиады.
Автоматическая проверка ответов для командных соревнований.
Подведение итогов в зависимости от скорости последнего ответа. (Кто раньше ответит, тот будет выше в сетке статистике)

Laravel 8.51
Bootstrap 4.6.1

# Place_CTF_PSUTI
Веб площадка для соревнований в ПГУТИ

Проект на Laravel с реализованной регистрацией пользователей для участия и подготовки к CTF
Планируется разработка полноценной админки, для того чтобы можно было добавлять свои задачи из веб интерфейса админа, чтобы в будущем использовать платформу вновь, как для соревнований, так и для подготовки участников своей команды.
